# SwiftShop - Mini E-Commerce Frontend

A Mini E-Commerce frontend built with **Next.js**, **React**, and **TypeScript**. This project demonstrates product listing, filtering, dynamic routing, and responsive design.

## Live Demo

[SwiftShop Live](https://quick-shop.vercel.app)

## Features

- **Product Listing Page**
  - Fetches products from **FakeStore API**
  - Displays product image, title, price, and short description
  - Clean and structured UI

- **Filtering**
  - Filter products by **price range**
  - Filter products by **category**
  - Dynamic filtering without page reload

- **Product Details Page**
  - Dynamic routing to show full product details
  - Proper **loading** and **error** handling

- **Responsive Design**
  - Works on mobile, tablet, and desktop screens

- **Code Structure**
  - Reusable components for maintainability
  - Proper folder structure
  - Clean and readable code

- **Bonus / Extras**
  - Styled using **Tailwind CSS**
  - Built with **TypeScript**
  - Project deployed on **Vercel**

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel (for deployment)
